# DataReact Profile

> **CV/Resume Editor with PDF Generation**

A Next.js application for creating and managing professional CVs with PDF export capabilities.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
